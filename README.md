# hubot-omi

A hubot script that connects HPE OMi

See [`src/omi.coffee`](src/omi.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-omi --save`

Then add **hubot-omi** to your `external-scripts.json`:

```json
[
  "hubot-omi"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
